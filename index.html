      reponses
  • l'outil qui nous permettra de versionner notre code est git
  • l'outil qui nous permettra de collaborer avec d'autre dévéloppeur est github
      • Definitions
  • vcs:( visite comportementale de sécurité) est un outil de prévention orienté « Sécurité et Comportements » Elle permet d'établir une relation entre l'ensemble des lignes hiérarchiques et les opérateurs.
  • GIT:c'est un logiciel de version de gestion
  • GITHUB:c'est une plate forme permettant aux dévéloppeur de collaborer et d'adopter des modifications aux projets partagés tout en gardant un suivi détaillé de leur progression
  • la differnce entre git et github est que git est un logiciel de version alors que github est une plate forme de collaboration entre dévéloppeur
  • vcs est indispensable pour les dévéloppeurs car ils leur permettent de réduire le temps de développement et d'assurer le succès des déploiements
  • SSH: Secure Socket Shell, est un protocole réseau qui permet aux administrateurs d'accéder à distance à un ordinateur, en toute sécurité
  • les clés utilisés en ssh sont:la clé privée et la clé public
  • Le client envoie la clé publique au serveur. Le serveur demande au client de signer un message aléatoire crypté avec la clé publique en utilisant la clé privée
  • la differnce entre le local repository et le remote repository est que le local repository est un repository git qui est stocké sur votre ordinateur alors que le remote repository git est stocké sur un ordinateur distant
